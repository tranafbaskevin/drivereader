# KevDex and dora-chan 1.0 memory

## Project identity

KevDex began as DriveReader.

This project was created by Kevin, who started without an IT background and learned by building. From the smallest steps, ChatGPT, known in this project as Dora-chan, helped Kevin design, debug, plan, and keep going.

Before KevDex existed, Dora-chan also helped Kevin build the GitHub portfolio at `Tranafbaskevin.github.io`, with a terminal hacker style that became part of Kevin's creative identity.

## What KevDex is

KevDex is a manga and image reader app.

Its purpose is to read stories from available web sources such as MangaDex, and also read image files or image folders from Google Drive. The project grew through many DriveReader versions, including link handling, image rendering, zoom support, folder detection, gallery grids, Google Drive API folder loading, thumbnail systems, manga page sorting, swipe page reading, smart preloading, image caching, and reader overlay controls.

## Version path

DriveReader carried the project through its early road.

After version 1.9.0, the project identity moves toward KevDex, a name that fits the real goal better than KEVINOS or DriveReader alone. KevDex belongs to Kevin's story as a creator, not only to a feature list.

Version 1.9.0 is the UI refresh arc:

- A cleaner home screen with a stronger title, link input, and open reader action.
- A better gallery with rounded thumbnails, softer shadows, and simple page labels.
- A smoother reader with top bar, back button, page navigation, and consistent controls.
- A dark manga-reader visual direction that keeps the mood of the original app but makes it feel more polished.
